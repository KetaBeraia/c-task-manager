
# c-task-manager

A simple file-based task manager written in C that allows adding, listing, and saving tasks to a file.

## Features
- Add a task with description and ID
- List all tasks
- Mark tasks as complete
- Save tasks to `tasks.txt` and load them on program start

## Requirements
- GCC or any C compiler
- Terminal / command line

## How to Build
```bash
gcc main.c -o task_manager
